# Support

## Getting Help

Need help with ReasonKit Mem? Here are your options:

### Documentation

- **Official Docs:** [docs.rs/reasonkit-mem](https://docs.rs/reasonkit-mem)
- **Website Guides:** [reasonkit.sh/docs](https://reasonkit.sh/docs)
- **Memory Architecture Guide:** [MEMORY_ARCHITECTURE_2025.md](docs/MEMORY_ARCHITECTURE_2025.md)

### Community Support

- **GitHub Discussions:** [Ask questions & discuss](https://github.com/reasonkit/reasonkit-mem/discussions)

### Bug Reports & Feature Requests

- **Bugs:** [Submit a bug report](https://github.com/reasonkit/reasonkit-mem/issues/new?template=bug_report.yml)
- **Features:** [Request a feature](https://github.com/reasonkit/reasonkit-mem/issues/new?template=feature_request.yml)

### Security Issues

For security vulnerabilities, please email **security@reasonkit.sh** instead of opening a public issue.

See [SECURITY.md](SECURITY.md) for our security policy.

### Commercial Support

For enterprise support, SLAs, or consulting, contact **enterprise@reasonkit.sh**.

---

## Response Times

| Channel          | Expected Response |
| ---------------- | ----------------- |
| Security issues  | < 24 hours        |
| Bug reports      | < 48 hours        |
| Feature requests | < 1 week          |
| Discussions      | Community-driven  |

## Before Asking

1. Search [existing issues](https://github.com/reasonkit/reasonkit-mem/issues)
2. Search [discussions](https://github.com/reasonkit/reasonkit-mem/discussions)
3. Check the [docs](https://docs.rs/reasonkit-mem)
4. Review [embedded mode guide](docs/EMBEDDED_MODE_GUIDE.md)
